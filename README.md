# Phrases api

`phrases` API from [open-korean-text](https://github.com/open-korean-text/open-korean-text) on the AWS Lambda.

## Requirements

- Java 8
- Gradle 6.5.1
- [Serverless framework](https://www.serverless.com/)

## Deployment

```bash
./gradlew build
sls deploy
```
